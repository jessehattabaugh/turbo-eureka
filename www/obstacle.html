Obstacle

Obstacle is an object like an image or a canvas which is skewed rotated and translated to give the appearance that the obstacle is moving around the player in 3D space as they move around the track. Obstacles are represented as Matter.js physics Bodies, and as such can be collided with. Obstacles can be placed manually using attributes.

This component wraps an array of images that will be scattered around the track for the player to avoid. Components that don't have a position will be repeated some default number of time. The repetitions will be configurable with an attribute.
